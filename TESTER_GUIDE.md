# Tester Guide

## Goal
Use the app freely for 15-20 minutes and provide brutally honest feedback.

## Test flow
1. Open the live app
2. Import backup
3. Explore dashboard
4. Browse portfolio
5. Run AI Scan flow
6. Review lots and box ROI
7. Add a watchlist target
8. Try moving target into portfolio

## Evaluate
- First impression
- Visual quality
- Ease of use
- Clarity of AI workflow
- Perceived usefulness
- Confusing moments
- Features that feel strong
- Features that feel unnecessary

## Final question
Would you genuinely use this for your own football card activity?
Why or why not?
