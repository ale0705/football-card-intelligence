# Roadmap

## Product vision
Build an intelligent, football-specific portfolio platform for collectors, investors and sellers of football trading cards.

The product should not become a generic card catalog. The core direction is football card intelligence: valuation support, portfolio decisions, box ROI, lots strategy, prospect monitoring and future event-driven alerts.

## Current foundation
- Local-first portfolio management
- AI-assisted card analysis workflow
- Box ROI tracking
- Lots builder
- Sold cards tracking
- Football layer: club, rookie, academy, prospect level
- Dashboard scenarios: prudent, realistic, optimistic
- Watchlist for future targets
- GitHub Pages live deployment

## Near-term priorities
1. Stabilize tester feedback loop
2. Polish Watchlist UX
3. Improve onboarding and tooltips
4. Refactor project structure safely
5. Prepare team-ready architecture

## Medium-term priorities
1. AI classification suggestions
   - Rookie probability
   - Academy monitored suggestion
   - Prospect level suggestion
   - Club and competition extraction
2. Academy Radar evolution
   - young players monitoring
   - event notes
   - review reminders
3. Watchlist intelligence
   - buy triggers
   - target price status
   - conversion to portfolio
4. Better seller workflow
   - listing preparation
   - lots pricing strategy
   - sold partial lots logic

## Long-term vision
1. Backend and user accounts
2. Cloud sync
3. One-click AI Scan
4. Marketplace comparison layer
5. Event-driven repricing
6. Mobile / PWA experience
7. Possible SaaS launch
